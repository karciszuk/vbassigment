# vbassigment
## Setup
### 1. Create virtual environment:
```
python -m venv .venv
```
### 2. Activate virtual environment:
#### Windows
```
.venv\Scripts\activate
```
#### Mac/Linux
```
source .venv/Scripts/activate
```
### 3. Install dependencies:
```
pip install -r requirements.txt
```

### 4. Run notebooks:
```
jupyter notebook
```
