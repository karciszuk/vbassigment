# vbassigment
## Setup
### 1. Create virtual environment:
```
python -m venv .venv
```
### 2. Activate virtual environment:
#### Windows
```
.venv\Scripts\activate
```
#### Mac/Linux
```
source .venv/Scriptspi/activate
```
### 3. Install dependencies:
```
pip install -r requirements.txt
```
### 4. Install project as editable package:
```
pip install -e .
```
### 5. Run notebooks:
```
jupyter notebook
```
